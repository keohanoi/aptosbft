# Contributors to AptosBFT

AptosBFT is developed by the Aptos Foundation and community contributors.

## Core Team

- **Aptos Foundation** - Project oversight and direction
- **Consensus Team** - Core algorithm development and verification

## Contributors

Thank you to all contributors who have helped make AptosBFT better!

### Contributing

See [`CONTRIBUTING.md`](CONTRIBUTING.md) for guidelines on how to contribute.

## Recognition Guidelines

Contributors are recognized for:

- **Code Contributions**: Pull requests that improve the codebase
- **Bug Reports**: Reporting bugs and helping verify fixes
- **Documentation**: Improving documentation and examples
- **Reviews**: Code reviews and constructive feedback
- **Testing**: Writing and improving test coverage

## How to Add Yourself

When you make your first merged contribution:

1. Your name will automatically be added to this file
2. You'll be recognized in the release notes
3. You'll be eligible for contributor badges (if applicable)
